UI/UX design/development

p5.js/Javascript/Github

In this project, we developed an engaging and visually appealing game using p5.js and JavaScript . We adopted an object-oriented programming (OOP) approach, creating multiple classes to encapsulate the game's mechanics and enhance modularity. This allowed for cleaner, more maintainable code and streamlined collaboration within our group.

Throughout the development process, we utilized GitHub for version control, ensuring a clear commit history that reflected our progress over time. This facilitated collaboration, as each member contributed meaningfully to the project. The game was designed to be interactive and entertaining, featuring dynamic visuals and smooth user interactions, achieved through the powerful capabilities of the p5.js library and the HTML canvas element. To showcase our work, we created a concise trailer that highlights the game's key features, further emphasizing the user experience
