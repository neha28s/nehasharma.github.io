Lunar Lander

UI/UX design/development

p5.js/Javascript/Github

In this Lunar Lander game project, I used JavaScript and Object-Oriented Programming to create an interactive moon landing experience. The game includes three main screens: a start screen, the game screen where the user can control the spacecraft's descent using the up-arrow key to activate the thrusters, and a result screen displaying the outcome of the landing. The objective is to safely land the spacecraft by managing its non-linear velocity under gravity. The game is fully replayable and showcases my skills in game state management, object-oriented programing, and Javascript.
