Web development

Javascript/Node.js/Handlebars/Express.js

This project is a one-page dynamic web application developed with Node.js, Express, Handlebars, and SQLite3, demonstrating seamless integration of backend and frontend technologies. The site includes a secure login system with hashed passwords.Admin has the right to perform CRUD operations in events table (create, read, update, delete) on database records. Featuring a paginated list view with detailed item access, it uses a relational database with linked tables and SQL queries. With a responsive design using Flexbox/Grid CSS, the site adapts to desktop, tablet, and mobile, while SQL injection prevention ensures secure user interactions.
