Figma/HTML/CSS/Javascript

For our final project, we developed a fully responsive e-commerce website, showcasing our skills in web design and development. The website includes a home page, a product listing page with at least ten items, and a product detail page. We implemented a complete order process, allowing users to add items to their shopping cart, proceed to checkout, and receive a payment confirmation.

Key features include interactive animations and JavaScript-based functionalities such as dynamic product filtering and a responsive navigation menu. We created the wireframes and a high-fidelity prototype in Figma for both mobile and desktop layouts, along with a design system for consistent visual styling.

The project was collaboratively managed on GitHub, with each team member contributing through frequent commits and code reviews. Our focus was on clean, well-structured code and a visually engaging user experience, demonstrating our proficiency in HTML, CSS, and JavaScript.

"This small project taught me a lot about working within tight timeframes while also ensuring the delivery of a functional website with user-friendly UI/UX design.

Co-creators: Nancy and Amber.
